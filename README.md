# Event Planner (React Capstone)

A simple event planning web app built with React, React Router, Formik, React-Bootstrap, and the Context API.

## Features
- Register and login (demo)
- Add, edit, and delete events
- Dashboard view of upcoming events
- Help page with guidance

## Tech Stack
- React + Vite
- React Router
- Formik
- Context API
- React-Bootstrap

## Install & Run
1. Install dependencies:
   npm install
